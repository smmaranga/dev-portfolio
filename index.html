Python

EnlistMe! Isko vs Iska

Java

EnlistMe! Isko vs Iska is a Java-based two-player shooting game developed as a requirement for CMSC 22, designed to apply Object-Oriented Programming (OOP) principles. Played on a single PC, players compete by shooting each other to gain points from power-ups while also decreasing their opponent’s score. Simple yet competitive, the game showcases fundamental OOP concepts in an engaging and interactive way.

Friendbook.

Dart, Flutter

Friendbook is a Flutter-based mobile app developed as a requirement for CMSC 23, modernizing the classic slambook experience. Users can create accounts, add friends by filling out a digital slambook form, or scan QR codes for quick connections. Serving as a personalized digital keepsake, it blends nostalgia with technology while applying key mobile app development concepts.

AgriLink

JavaScript, ReactJS, Tailwind CSS, HTML

AgriLink is a farm-to-table website developed as an academic requirement for CMSC 100. It streamlines the process of connecting local farmers with consumers, ensuring fresh, high-quality produce is easily accessible. The platform features a dual-interface system, catering to both customers and administrators for a seamless ordering experience.
